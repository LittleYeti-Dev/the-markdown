## Key Findings

1. **Meta key mismatch (BF02):** Snippet 74 queried `_ns_promoted = 1` but actual meta keys use no `_ns_` prefix — the correct key is `promote_status` with value `dev`. Fix was to remove the meta_query entirely so all feed items appear in RSS.

2. **WordPress Customizer hide-title (BF05):** The Customizer "hide title" setting applies `display: none !important` on `.entry-title` AND `height: 0px` on `.entry-header`. Even inline styles cannot override. Solution: inject headline via `the_content` filter and hide the theme's `.entry-header` with CSS.

3. **Custom feed architecture (BF02):** `/feed/the-markdown/` uses `add_feed()` with its own `get_posts()` — `pre_get_posts` hooks are ineffective. Must patch the source snippet directly.

4. **Pagination explosion (BF03):** The `for` loop rendered one button per page (83+). WordPress's `paginate_links()` with `end_size: 1, mid_size: 2` constrains to ~7-9 buttons with ellipsis.
